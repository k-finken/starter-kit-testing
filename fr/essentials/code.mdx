## Code inline

Pour désigner un `mot` ou une `phrase` comme code, encadrez-le avec des backticks.

## Blocs de code

Utilisez des blocs de code délimités en entourant le code de trois backticks, suivi du langage de programmation pour la coloration syntaxique.
