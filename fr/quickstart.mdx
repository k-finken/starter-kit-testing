## Commencez en trois étapes

Lancez votre site de documentation localement et faites votre première personnalisation en quelques minutes.

### Étape 1 : Configurer votre environnement local

<AccordionGroup>
  <Accordion icon="copy" title="Cloner vos docs localement">
    Pendant l'intégration, vous avez créé un dépôt GitHub avec le contenu de vos docs. Vous pouvez trouver le lien dans votre [tableau de bord](https://dashboard.mintlify.com).

    Pour cloner le dépôt localement, suivez le guide [Cloner un dépôt](https://docs.github.com/fr/repositories/creating-and-managing-repositories/cloning-a-repository) dans la documentation GitHub.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Démarrer le serveur de prévisualisation">
    1. Installez le CLI Mintlify : `npm i -g mint`
    2. Accédez à votre répertoire docs et exécutez : `mint dev`
    3. Ouvrez `http://localhost:3000` pour voir vos docs en direct !

    <Tip>Votre prévisualisation se met à jour automatiquement lorsque vous modifiez les fichiers.</Tip>
  </Accordion>
</AccordionGroup>

### Étape 2 : Déployer vos changements

<AccordionGroup>
  <Accordion icon="github" title="Installer notre application GitHub">
    Installez l'application GitHub Mintlify depuis votre [tableau de bord](https://dashboard.mintlify.com/settings/organization/github-app).

    Notre application GitHub déploie automatiquement vos changements sur votre site de documentation.
  </Accordion>
  <Accordion icon="palette" title="Mettre à jour le nom et les couleurs">
    Pour un premier changement, mettons à jour le nom et les couleurs de votre site.

    1. Ouvrez `docs.json` dans votre éditeur.
    2. Modifiez le champ `"name"` avec le nom de votre projet.
    3. Mettez à jour les `"colors"` pour correspondre à votre marque.
    4. Enregistrez et voyez vos changements instantanément sur `http://localhost:3000` !

    <Tip>Essayez de changer la couleur principale pour voir une différence immédiate !</Tip>
  </Accordion>
</AccordionGroup>

### Étape 3 : Mise en ligne

<Accordion icon="rocket" title="Publier vos docs">
  1. Commitez et poussez vos changements.
  2. Vos docs seront mis à jour et en ligne en quelques instants !
</Accordion>
