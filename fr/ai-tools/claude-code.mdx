Configurez Claude Code pour vous aider à rédiger et maintenir votre documentation Mintlify.

## Prérequis

- Abonnement Claude actif (Pro, Max ou accès API)

## Configuration

<Steps>
<Step title="Installer Claude Code">
  ```bash
  npm install -g @anthropic-ai/claude-code
  ```
</Step>

<Step title="Installer la compétence Mintlify">
  Dans votre répertoire docs, exécutez :

  ```bash
  npx skills add https://mintlify.com/docs
  ```
</Step>
</Steps>
