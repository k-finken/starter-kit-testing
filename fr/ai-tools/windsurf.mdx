Configurez l'assistant IA Cascade de Windsurf pour vous aider à rédiger et maintenir votre documentation Mintlify.

## Prérequis

- Windsurf installé

## Configuration

<Steps>
<Step title="Ouvrir votre répertoire docs dans Windsurf">
  Ouvrez la racine de votre dépôt de documentation où se trouve `docs.json`.
</Step>

<Step title="Installer la compétence Mintlify">
  Dans le terminal intégré, exécutez :

  ```bash
  npx skills add https://mintlify.com/docs
  ```
</Step>
</Steps>
